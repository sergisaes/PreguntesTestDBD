  built to make revising fast and self-graded.
</p>

---

## What it is

A small no-build web app (plain HTML/CSS/JS) that turns exam-style questions into an interactive
quiz: answer, get instant feedback, and track how you're doing — so you can drill weak spots instead
of re-reading notes. Made for myself and coursemates to study more efficiently.

## Run it

Just open `index.html` in a browser — no server, no build step.

## Stack

HTML · CSS · vanilla JavaScript
